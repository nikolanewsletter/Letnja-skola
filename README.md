# Letnja skola

Next.js aplikacija za projekat Letnja skola.

## Getting Started

Pokreni development server:

```bash
npm run dev
```

Otvori [http://localhost:3000](http://localhost:3000) u browseru.

Glavna stranica je u `src/app/page.tsx`.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- ESLint

## Deploy

Najjednostavniji deploy za Next.js je preko Vercel platforme.
